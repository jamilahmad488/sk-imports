# SK Imports Website

Static GitHub Pages website for SK Imports — Mobile Accessories & Electronic Gadgets.

## Pages
- Home: `index.html`
- Products: `products.html`
- Shop: `shop.html`
- About: `about.html`
- Contact: `contact.html`

## How to Use
1. Upload all files to your GitHub repository.
2. Go to Repository Settings.
3. Open Pages.
4. Select branch: `main`.
5. Select folder: `/root`.
6. Save.
7. Your website will be live on GitHub Pages.

## Image Replacement
Add your final logo as:

`assets/images/logo.png`

Then replace the placeholder area in `index.html` with:

```html
<img src="assets/images/logo.png" alt="SK Imports Logo" class="hero-logo">
```
